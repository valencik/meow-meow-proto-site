# Cats Effect 3.7 Released

*December 12, 2025*

We're excited to announce the release of Cats Effect 3.7, bringing new features and performance improvements to the ecosystem.

## Highlights

This release includes several notable improvements:

- **Improved Fiber Scheduling** — Better fairness and reduced latency
- **New Tracing APIs** — Enhanced debugging capabilities
- **Performance Optimizations** — Faster IO operations

## Migration

Upgrading from 3.6.x should be seamless for most users. Check out the [migration guide](https://typelevel.org/cats-effect/) for details.
