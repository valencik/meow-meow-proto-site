# Community

Join the Typelevel community! We're an inclusive group of developers passionate about functional programming in Scala.

## Where to Find Us

- **Discord** — Our primary community chat: [Join Discord](https://discord.gg/XF3CXcMzqD)
- **GitHub** — Contribute to our projects: [github.com/typelevel](https://github.com/typelevel)
- **Mastodon** — Follow us: [@typelevel](https://fosstodon.org/@typelevel)

## Code of Conduct

We are committed to providing a friendly, safe and welcoming environment for all, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, sexual identity and orientation, or other such characteristics.

[Typelevel Code of Conduct](CODE-OF-CONDUCT.md)

## Getting Help

- Ask questions on Discord in the `#beginners` channel
- Open issues on the relevant GitHub repository
- Check the documentation for each project

## Contributing

We welcome contributions! Each project has its own contributing guidelines. A good place to start is issues labeled "good first issue" on our repositories.
